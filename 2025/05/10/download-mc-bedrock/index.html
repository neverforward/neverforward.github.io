Minecraft 国际基岩版下载教程

neverforward2025-05-102025-05-14

前言

众所周知，刚毅mc优化逆天。解决办法很简单，就是下载国际版。

电脑版下载

下载

微软商店（不推荐，网速慢，要正版89块）

首先，打开 微软商店 ，在搜索栏搜索 Minecraft。点击 Minecraft: Java Edition & Bedrock Edition for PC

然后点击下载，等待下载完成。接着点击开始游戏即可。

MCAPPX（推荐，需要破解）

打开 MCAPPX 版本库。

引用站外地址
MCAPPX 版本库

如图所示：

然后，点击立即下载按钮。等待下载完成。

下载完成后，点击打开文件。弹出窗口。按着提示点下一步即可。

这里我已经安装完了，所以显示重新安装。

如果你安装完了，点击开始菜单 ，再点击全部 ，在里面找到 M ，再点击Minecraft即可。

破解

如果你购买了正版，请忽略这步。

首先，下载 Unlock Minecraft。

引用站外地址
Unlock Minecraft

下载之后解压文件，如图：

如果你是64位系统，打开 64x；否则打开 32x。

之后，运行 M-Centres 3.0.exe。

点击 Install Mode，再点击 Start Operation。等待上方出现 Success 字样。

打开 Minecraft，即可。

手机版下载（苹果手机不行）

打开 mcapks.com

引用站外地址
mcapks.com

如图所示：

选择最新正式版，点击下载 -> 下载游戏。

下载完成后，点击安装。

安装后打开游戏即可

结言
