1002 - 编程求解1+2+3+...+n

题目描述

使用编程求解下列式子的值： 𝑆=1+2+3+...+𝑛

输入

输入一行，只有一个整数 𝑛(1≤𝑛≤1000)

输出

输出只有一行（这意味着末尾有一个回车符号），包括1 个整数。

code:


#include<iostream>
using namespace std;

int main() {
	int r{0},a{0};
	cin >> a;
	for(int i=0;i<=a;i++){
		r+=i;
	}
	cout << r << endl;
	return 0;
}

1004 - 编程求1*2*3*...*n

题目描述

编程求 1×2×3×⋯×𝑛。

输入

输入一个正整数𝑛。

输出

输出只有一行（这意味着末尾有一个回车符号），包括1个整数。

code:


#include<iostream>
using namespace std;  

int main() {  
    int r = 1, a = 0;  
    cin >> a;  
    for (int i = 1; i <= a; i++) {  
        r *= i;
    }  
    cout << r << endl;  
    return 0;  
}

1005 - 编程求解：已知一个圆的半径，求解该圆的面积和周长

题目描述

已知一个圆的半径，求解该圆的面积和周长。

输入

输入一行，只有一个整数

输出

输出只有两行，一行面积，一行周长。（保留两位小数）。𝑝𝑖=3.1415926。

code:


#include <stdio.h>
#include <math.h>  

int main() {  
    double r, S, C;   
    scanf("%lf", &r);  
    S = M_PI * r * r;  
    C = 2 * M_PI * r;  
    printf("%.2f\n", S);  
    printf("%.2f\n", C);  
    return 0;  
}
